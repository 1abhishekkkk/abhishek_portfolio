# Showreel Videos

Place your showreel videos in this folder.

## File Requirements:
- **Format**: MP4 (H.264 codec recommended)
- **Aspect Ratio**: 9:16 (vertical/portrait) for best display
- **Duration**: 15-60 seconds recommended
- **Size**: Under 50MB per video for web performance
- **Quality**: 1080x1920px or similar vertical resolution

## File Naming:
- `reel1.mp4` - Your first/best reel
- `reel2.mp4` - Second reel
- `reel3.mp4` - Third reel
- etc.

## Video Tips:
- Keep audio levels consistent across all reels
- Start with your strongest/most engaging content
- Ensure videos work well both muted and with sound
- Consider adding subtitles for accessibility
