# 3D Models Folder

Place your 3D avatar models here in GLB or GLTF format.

## Recommended File Names:
- `avatar.glb` - Your main avatar model
- `avatar-casual.glb` - Casual pose variant
- `avatar-professional.glb` - Professional pose variant

## File Requirements:
- Format: GLB (preferred) or GLTF
- Size: Under 5MB for optimal web performance
- Textures: Embedded in GLB format

## Usage Example:
```jsx
<HeroAvatar modelPath="/models/avatar.glb" />
<FloatingAvatar modelPath="/models/avatar-casual.glb" />
```
